Virtual Touching Project คืออะไร?

Virtual Touching Project คือกลุ่มของ ทาเล้นท์ และผู้สร้างคอนเทนต์ที่มารวมตัวกันเพื่อถ่ายทอดความรู้สึกผ่านการไลฟ์สตรีม ที่นี่เป็นพื้นที่สำหรับการเชื่อมต่อระหว่างผู้ชมกับ VTuber ผ่านอารมณ์ ความคิด และเรื่องราวที่อยากแบ่งปัน เราหวังว่าคุณจะได้พบกับทาเล้นท์ที่คุณชื่นชอบ และร่วมเป็นส่วนหนึ่งของช่วงเวลาที่มีความหมายไปด้วยกัน

สำหรับผู้ที่สนใจเข้าร่วมกับเรา

หากคุณสนใจเป็นส่วนหนึ่งของ Virtual Touching Project ไม่ว่าจะในฐานะ ทาเล้นท์ หรือ VTuber เรามีแบบฟอร์มให้คุณสามารถส่งคำขอเข้าร่วมเบื้องต้นได้อย่างสะดวก
