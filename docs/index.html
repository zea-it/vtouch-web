Virtual Touching Project (Vtouch)

Vtouch Project คือ "กลุ่มก้อน" ของเหล่าทาเลนต์และ V-Tuber ที่รักอิสระ ซึ่งมารวมตัวกันเพื่อสร้างสรรค์คอนเทนต์และแบ่งปันเรื่องราว

แทนที่จะเป็นสังกัดในรูปแบบบริษัท เราคือพื้นที่ที่ให้ความสำคัญกับอิสระและความเป็นตัวของตัวเอง ที่นี่คือจุดเชื่อมต่อระหว่างผู้ชมและครีเอเตอร์ ผ่านไลฟ์สตรีม, ผลงาน, และ "ความรู้สึก" ที่เราอยากส่งต่อให้กัน เราหวังว่าคุณจะได้พบกับทาเลนต์ที่ชื่นชอบ และร่วมเป็นส่วนหนึ่งในช่วงเวลาดีๆ ไปด้วยกัน

สำหรับผู้ที่สนใจเข้าร่วมกับเรา

หากคุณสนใจเป็นส่วนหนึ่งของ Virtual Touching Project ไม่ว่าจะในฐานะ ทาเลนต์ หรือ VTuber เรามีแบบฟอร์มให้คุณสามารถส่งคำขอเข้าร่วมเบื้องต้นได้อย่างสะดวก
