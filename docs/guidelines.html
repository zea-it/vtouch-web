ปรัชญาและแนวทางการทำงาน

เอกสารนี้จัดทำขึ้นเพื่อสร้างความเข้าใจที่ตรงกัน เกี่ยวกับรูปแบบการทำงานที่เป็นเอกลักษณ์ของ Vtouch Project

คำชี้แจง: Vtouch Project ไม่ใช่บริษัท หรือสตูดิโอขนาดใหญ่

Vtouch Project ดำเนินการโดยบุคคลเพียงคนเดียวในทุกขั้นตอน ไม่ว่าจะเป็น:

  • **การออกแบบโมเดล**
  • **การทำ Live2D**
  • **งานกราฟิก Overlay**
  • **การตัดต่อวิดีโอ PV**
  • **หรือองค์ประกอบอื่น ๆ ที่เกี่ยวข้องกับผลงานทั้งหมดในสังกัด**

ด้วยเหตุนี้ รูปแบบการทำงานของเราจึงมีความแตกต่างจากองค์กรขนาดใหญ่ทั่วไป และมีข้อจำกัดที่ผู้ร่วมงานจำเป็นต้องทำความเข้าใจ

แนวทางการทำงาน

ดำเนินการโดยบุคคลเดียว

ทุกผลงานในสังกัดดำเนินการโดยบุคคลเพียงคนเดียว ไม่ได้มีทีมงานหรือแผนกสนับสนุน จึงไม่สามารถกำหนดวันแล้วเสร็จของงานได้อย่างแน่นอน และทุกกระบวนการขึ้นอยู่กับกำลังความสามารถของบุคคลนั้น

ทำงานตามจังหวะและเวลาที่เอื้ออำนวย

งานจะถูกดำเนินการในช่วงเวลาที่ผู้ดำเนินการมีความพร้อมทางกายภาพและจิตใจ ไม่สามารถทำงานตามตารางเวลาที่ตายตัว หรือเร่งรัดงานได้ในทุกกรณี เราให้ความสำคัญกับคุณภาพของผลงานและความสมดุลในการทำงาน

การอัปเดตงานไม่เป็นระยะที่แน่นอน

จะมีการอัปเดตความคืบหน้าของงานเมื่อมีความคืบหน้าอย่างเป็นรูปธรรมเท่านั้น ไม่ได้อัปเดตทุกสัปดาห์หรือตามรอบเวลาตายตัว เพื่อให้ข้อมูลที่แม่นยำและเป็นประโยชน์ที่สุด

ข้อตกลงสำหรับผู้ที่สนใจร่วมงาน

เพื่อให้การทำงานร่วมกันเป็นไปอย่างราบรื่นและมีความเข้าใจตรงกัน โปรดอ่านและยอมรับแนวทางข้างต้นให้ชัดเจนก่อนตัดสินใจร่วมงาน:

  • **ต้องสามารถยอมรับการรอคอย** ที่ไม่สามารถระบุกรอบเวลาได้อย่างแน่นอน
  • **ขอความร่วมมือในการสื่อสารด้วยความเข้าใจและเคารพซึ่งกันและกัน**
  • **หลีกเลี่ยงการเร่งรัดหรือกดดัน** เพราะจะส่งผลต่อคุณภาพและสภาพจิตใจของผู้ดำเนินการโดยตรง

เป้าหมายของ Vtouch Project

Vtouch Project ก่อตั้งขึ้นเพื่อเป็นพื้นที่ทดลองและพัฒนา ไม่ใช่พื้นที่ของความสมบูรณ์แบบ แต่เป็นพื้นที่ของการเรียนรู้ เติบโต และทำสิ่งที่รักในขอบเขตที่สามารถทำได้จริง ด้วยความตั้งใจที่แท้จริง ไม่ใช่เพื่อแสวงหากำไรหรือผลประโยชน์ในเชิงธุรกิจ

หากท่านเข้าใจและยอมรับแนวทางนี้ Vtouch Project ยินดีต้อนรับเสมอ พร้อมที่จะสร้างสรรค์สิ่งดี ๆ ไปด้วยกัน

พร้อมที่จะร่วมเป็นส่วนหนึ่งของเราแล้วหรือยัง?
